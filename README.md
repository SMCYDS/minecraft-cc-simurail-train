# CC+Aeronautics 定速 PID 巡航机车控制系统

基于 **ComputerCraft** 与 **Create Aeronautics** 模组，实现机车（飞艇/列车）的自动定速巡航控制。系统采用 **PID 控制算法**，通过调节模拟油门拉杆，使机车实际速度稳定在设定的目标速度，并支持多种操控方式（直接设定 / 升降保控制）。

---

## 计算机输入输出配置(在config配置信号面)
*注：建议搭配使用Drive By Wire(线控)模组
| 栏目 | 信号类型 | 作用 |
|--------|------|
| 定速模拟输入 | **模拟红石信号** | 0~15分别映射到0~Vmax的速度 |
| 换端信号输入 | **二进制信号** | 建议使用拉杆输入 |
| 速度传感器 | **外设读取** | 用于读取实时速度信息 |
| 刹车模拟输出 | **模拟红石信号** | 0~15分别映射到不同刹车强度，config中可以调整刹车比例防止力度过大 |
| 油门模拟输出 | **(反向)模拟红石信号** | 15~0反向映射到油门大小(直接供模拟齿轮箱使用) |

## 文件结构

| 文件名 | 作用 |
|--------|------|
| `config_swd1p.lua` | **配置文件**：存储机车属性（最大速度、PID 参数、信号输入/输出方向等），供主程序加载。 |
| `loco.lua` | **主运行程序**：启动后持续读取速度反馈，执行 PID 运算，并输出油门控制信号。 |
| `loco_rhi.lua` | **升降保控制适配的主运行程序**：将红石信号强度（0~15）转换为目标速度的升降/保持指令，方便外部红石设备（如按钮、拉杆）进行远程调速。 |
| `swd1p_s_cc.nbt` | **结构蓝图文件**（可选）：包含机车上 ComputerCraft 电脑、显示器、红石接口的预设布局，可直接粘贴到游戏中。需要安装线控模组（drive by wire）以确保正常控制 |

---

## 功能特性

- **变增益PID控制**：不同加速阶段改变Kp值，克服恒定误差问题
- **红石信号集成**：
  - 可从**指定方向**读取红石模拟信号作为目标速度输入（如来自控制面板的电位器）。
  - 可向**指定方向**输出模拟油门和刹车信号（0~15），驱动模拟齿轮箱和刹车。
- **状态显示**（需搭配显示器）：可实时显示当前速度、目标速度、PID 输出值，便于调试。
- **安全限速**：配置文件可设定最大允许速度。
- **降保升模式**：更贴近现实驾驶环境

## ⚙️ 配置说明（`config_swd1p.lua`）

配置文件是一个 Lua 表，返回各项参数。你需要根据你的机车实际布局修改以下字段：

```lua
-- config_swd1p.lua 示例（注：上传到cc电脑请使用库中英文注释版代码，防止乱码）
return {
    -- 最高速度（m/s），红石15对应此值，0对应停机
    maxSpeed = 25.0,

    -- ===== 红石接口方向 =====
    -- 输入
    sideSetSpeed  = "top",    -- 模拟输入：目标速度（0~15）
    sideReverse   = "right",  -- 数字输入：倒车信号

    -- 输出
    sideMonitor   = "left",   -- 显示器外设所在面
    sideBrake     = "bottom", -- 模拟输出：制动强度（0~15）
    sideThrottle  = "back",   -- 模拟输出：油门强度（通常反向输出）
    sideRevGearbox = "left",  -- 数字输出：倒车变速箱激活

    -- ===== RHI 升降保步进 =====
    -- 每 tick 速度变化量（m/s），1档（减速）为6倍，6档为1倍；9档（加速）为1倍，15档为7倍
    -- 数值越小调速越精细，响应越慢；越大则粗犷但响应快
    rhiStep = 0.2,

    -- ===== 控制循环间隔 =====
    pidDt = 0.1,   -- 秒，需与 loco.lua 中的 sleep() 一致

    -- ===== 积分与微分系数 =====
    pidKi = 0.6,   -- 积分：消除稳态误差
    pidKd = 0.4,   -- 微分：抑制超调

    -- ===== 变增益比例系数（按误差分区） =====
    -- 近区（|误差| < nearThreshold）：柔和，避免超调
    pidKpNear = 1.0,
    -- 中区（nearThreshold <= |误差| < farThreshold）：正常
    pidKpMid  = 1.5,
    -- 远区（|误差| >= farThreshold）：激进，快速追赶
    pidKpFar  = 3.0,
    nearThreshold = 0.8,  -- m/s，近/中边界
    farThreshold  = 5.0,  -- m/s，中/远边界

    -- ===== 死区 =====
    -- 误差小于此值则停止调节，防止在目标点反复开关油门/刹车
    pidDeadband = 0.1,  -- m/s，设为0则禁用

    -- ===== 刹车输出缩放 =====
    -- 范围 (0,1]，仅缩放输出信号，不影响PID误差计算
    brakeScale = 0.2,

    -- ===== 紧急刹车 =====
    -- 当目标速度为0时，根据当前速度 * 此系数施加刹车，独立于PID
    emergencyBrakeKp = 1.5,
}
